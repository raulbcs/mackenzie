Sugestão: Aplique uma engenharia reversa (você sabe o que é engenharia reversa?) E faça um diagrama de classes e o diagrama de sequência para o método Extrato da classe Cliente.

Aponte os problemas encontrados a partir desta análise.

Artefato de Entrega: Diagrama de Classes e Diagrama de Sequência
